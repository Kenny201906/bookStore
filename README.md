## 特性

- **uni-app**：一个使用 Vue.js开发所有前端应用的框架
- **Vue2**：采用渐进式的JavaScript框架
- **Uview UI**：是全面兼容nvue的uni-app生态框架
- **Vuex**: 状态管理工具
- **NPM**：快速的包管理工具
- **Sass**：世界上最成熟、稳定和强大的专业级CSS扩展语言
- **ESlint**：代码校验
- **Prettier**：代码格式化
- **GoEasy**：强大易用的websocket消息推送服务

## Git 提交规范

- `feat` 增加新功能
- `fix` 修复问题/BUG
- `style` 代码风格相关无影响运行结果的
- `perf` 优化/性能提升
- `refactor` 重构
- `revert` 撤销修改
- `test` 测试相关
- `docs` 文档/注释
- `chore` 依赖更新/脚手架配置修改等
- `workflow` 工作流改进
- `ci` 持续集成
- `types` 类型定义文件更改
- `wip` 开发中
- `mod` 不确定分类的修改
